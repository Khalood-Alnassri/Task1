-What is a database?
A database is like a digital storage system where information is kept in an orderly way so computers can quickly find, add, change, or delete data.

-Why databases are used?
Main reasons why databases are used:
1.Store large amounts of data: Databases can store huge volumes of information in an organized way.
2.Quick data access: Data can be searched and retrieved very fast.
3.Easy data management: Data can be added, updated, or deleted easily.
4.Data accuracy and consistency: Databases reduce errors and keep data correct and consistent.
5.Data security: Databases protect data using passwords and access controls.
6.Data sharing: Multiple users can access data at the same time.
7.Data backup and recovery:Databases allow data to be backed up and restored if lost.

-How databases reduce errors?
Databases reduce errors by checking, restricting, and automating data entry and updates.
